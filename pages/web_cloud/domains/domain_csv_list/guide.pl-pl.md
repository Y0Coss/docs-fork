## Wprowadzenie

Chcesz pobrać listę wszystkich domen dostępnych na Twoim koncie klienta OVHcloud? Chcesz odzyskać wszystkie informacje związane z Twoją domeną (przypisane kontakty, właściciel domeny, aktywne opcje, daty utworzenia i wygaśnięcia usług, etc.) w tym samym czasie?

**Dowiedz się, jak wyeksportować listę Twoich domen i powiązane z nimi informacje w formacie CSV z Panelu klienta OVHcloud.**

## Wymagania początkowe

- Posiadanie [domeny](/links/web/domains) zarejestrowanej w OVHcloud.
- Dostęp do [Panelu klienta OVHcloud](/links/manager).

## W praktyce

Kliknij poniższe zakładki, aby wyświetlić kolejne kroki **4**.

> [!tabs]
> **Etap 1**
>>
>> Zaloguj się do [Panelu klienta OVHcloud](/links/manager) i przejdź do sekcji `Web Cloud`{.action}.
>>
>> ![Web Cloud](/pages/assets/screens/control_panel/product-selection/web-cloud.png){.thumbnail}
>>
> **Etap 2**
>>
>> Kliknij menu `Domeny`{.action}. Nad tabelą, która się wyświetli kliknij przycisk `Eksportuj w formacie CSV`{.action}.
>>
>> ![Domain Names](/pages/assets/screens/control_panel/product-selection/web-cloud/domain-names.png){.thumbnail}
>>
> **Etap 3**
>>
>> W wyświetlonym oknie zaznacz informacje, które chcesz dołączyć do pliku CSV dla wszystkich Twoich domen.
>>
>> ![Export domains in CSV format step 1](/pages/assets/screens/control_panel/product-selection/web-cloud/domain-dns/export-csv/export-domains-in-csv-format-step-1.png){.thumbnail}
>>
>> Następnie kliknij przycisk `Eksport`{.action}, aby rozpocząć generowanie pliku w formacie CSV.
>>
> **Etap 4**
>>
>> Pojawi się okno ładowania. Zaczekaj, aż pasek postępu osiągnie 100%.
>>
>> ![Export domains in CSV format step 2 in progress](/pages/assets/screens/control_panel/product-selection/web-cloud/domain-dns/export-csv/export-domains-in-csv-format-step-2a.png){.thumbnail}
>>
>> Po wygenerowaniu pliku CSV automatycznie pojawi się następujący komunikat:
>>
>> ![Export domains in CSV format step 2 done](/pages/assets/screens/control_panel/product-selection/web-cloud/domain-dns/export-csv/export-domains-in-csv-format-step-2b.png){.thumbnail}
>>
>> Jeśli pobieranie pliku CSV nie rozpoczęło się automatycznie, kliknij link `pobrać plik ręcznie`{.action}, aby pobrać plik.
