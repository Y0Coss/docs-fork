## Objective

Want to retrieve a list of all the domain names on your OVHcloud customer account? Do you need to retrieve all information related to your domain names (associated contacts, holder, active options, creation and expiry dates, etc.) in one go?

**Find out how to export a list of your domain names and their associated information in CSV format from the OVHcloud Control Panel.**

## Requirements

- A [domain name](/links/web/domains) registered with OVHcloud.
- Access to your [OVHcloud Control Panel](/links/manager)

## Instructions

Click on the tabs below to view each of the **4** steps.

> [!tabs]
> **Step 1**
>>
>> Log in to the [OVHcloud Control Panel](/links/manager), then go to the `Web Cloud`{.action} section.
>>
>> ![Web Cloud](/pages/assets/screens/control_panel/product-selection/web-cloud.png){.thumbnail}
>>
> **Step 2**
>>
>> Click the `Domain names`{.action} menu. Above the table that appears, click `Export in CSV format`{.action}.
>>
>> ![Domain Names](/pages/assets/screens/control_panel/product-selection/web-cloud/domain-names.png){.thumbnail}
>>
> **Step 3**
>>
>> In the window that opens, tick the information that you want to include in the CSV file for all of your domain names.
>>
>> ![Export domains in CSV format step 1](/pages/assets/screens/control_panel/product-selection/web-cloud/domain-dns/export-csv/export-domains-in-csv-format-step-1.png){.thumbnail}
>>
>> Then click the `Export`{.action} button to start generating the file in CSV format.
>>
> **Step 4**
>>
>> A load window will appear. Wait until the progress bar reaches 100%.
>>
>> ![Export domains in CSV format step 2 in progress](/pages/assets/screens/control_panel/product-selection/web-cloud/domain-dns/export-csv/export-domains-in-csv-format-step-2a.png){.thumbnail}
>>
>> As soon as the CSV file is generated, the following message is displayed, and the file is automatically downloaded:
>>
>> ![Export domains in CSV format step 2 done](/pages/assets/screens/control_panel/product-selection/web-cloud/domain-dns/export-csv/export-domains-in-csv-format-step-2b.png){.thumbnail}
>>
>> If the CSV file download did not start automatically, click the `download it manually`{.action} link to retrieve it.
