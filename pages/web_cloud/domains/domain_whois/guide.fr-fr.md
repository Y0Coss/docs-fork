## Objectif

Vous souhaitez savoir si un nom de domaine est disponible en création ou non ? Vous désirez connaître certaines informations (coordonnées du titulaire, date d'expiration, etc.) relative à un nom de domaine ?

Le WHOIS, abréviation de l'anglais « Who is ? », est un protocole permettant d’accéder à différentes informations liées à un nom de domaine.

**Découvrez comment utiliser l'outil WHOIS pour vous renseigner sur un nom de domaine**

## Prérequis

- Disposer d'un appareil avec un navigateur Internet installé dessus (Firefox, Chrome, Safari, Edge, etc.).

## En pratique

OVHcloud met à disposition une interface pour vous permettre de lancer un WHOIS sur un nom de domaine.

> [!primary]
>
> L'affichage ou l'organisation des informations présentes dans le WHOIS peut être différent en fonction de l'extension du nom de domaine (par exemple : *.com*, *.net*, *.ovh*, etc.).
>
> De plus, les registres qui gèrent ces extensions peuvent permettre aux titulaire des noms de domaine de masquer dans le WHOIS certaines informations les concernant.

Pour utiliser le WHOIS sur un nom de domaine, effectuez les étapes suivantes :

1. Cliquez sur [ce lien](/links/web/domains-whois) pour accéder à notre page de recherche contenant notre outil WHOIS.
2. Sur la page qui s'affiche, renseignez le nom de domaine (par exemple : *domain.tld* **sans** les *www*) dans le champ `Nom de domaine *`{.action}, puis saisissez le `Code de sécurité` qui s'affiche dans le champ `Entrer le code de sécurité *`{.action}.
3. Cliquez sur le bouton `Whois`{.action} pour lancer la recherche.
4. En dessous du bouton `Whois`{.action}, la nouvelle section intitulée `Whois search results :`{.action} apparaît. Cette section contient le résultat du WHOIS pour le domaine que vous avez renseigné.

C'est dans ce résultat de recherche que vous trouverez toutes les informations disponibles sur le nom de domaine renseigné lors de l'étape **2** située plus haut dans ce guide.
