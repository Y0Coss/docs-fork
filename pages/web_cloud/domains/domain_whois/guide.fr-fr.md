## Objectif

Vous souhaitez savoir si un nom de domaine est disponible ? Vous désirez connaître certaines informations relatives à un nom de domaine, comme les coordonnées du titulaire, la date d'expiration, etc. ?

Le WHOIS, abréviation de l'anglais « Who is ? », est un protocole permettant d’accéder à différentes données liées à un nom de domaine.

**Découvrez comment utiliser l'outil WHOIS pour obtenir des informations sur un nom de domaine.**

## Prérequis

- Disposer d'un navigateur Internet (par exemple : Firefox, Chrome, Safari, Edge, etc.) sur votre ordinateur, votre tablette ou votre smartphone.

## En pratique

OVHcloud met à disposition une interface pour vous permettre d'exécuter la commande WHOIS pour un nom de domaine.

> [!primary]
>
> Le contenu et la présentation des informations dans le résultat d'une requête WHOIS peuvent différer en fonction de l'extension du nom de domaine (par exemple : *.com*, *.net*, *.ovh*, etc.).
>
> De plus, les registres qui gèrent ces extensions peuvent permettre aux titulaires des noms de domaine de masquer certains éléments personnels dans le WHOIS.

Les étapes suivantes permettent d’accéder aux enregistrements WHOIS associés à un nom de domaine :

1. Accédez à notre [outil WHOIS en ligne](/links/web/domains-whois).
1. Sur la page qui s'affiche, renseignez le nom de domaine (par exemple : *domain.tld* **sans** les *www*) dans le champ `Nom de domaine *`{.action}, puis saisissez le `Code de sécurité` qui s'affiche dans le champ `Entrer le code de sécurité *`{.action}.
1. Cliquez sur le bouton `Whois`{.action} pour envoyer la requête.
1. Consultez le résultat de la requête WHOIS dans la nouvelle section intitulée `Whois search results :`{.action} située sous du bouton `Whois`{.action}.

Dans ce résultat, vous retrouverez toutes les informations disponibles sur le nom de domaine renseigné lors de l'étape **2**.
