## Contexte : fin du réseau cuivre

Le réseau cuivre historique utilisé pour les connexions xDSL (ADSL, VDSL) est progressivement fermé en France par l’opérateur d’infrastructure. Ce plan, piloté par l’ARCEP et Orange, vise à généraliser l’adoption de la fibre optique (FTTH) et à moderniser les infrastructures télécoms.

### Pourquoi cette fermeture ?
- **Obsolescence du réseau cuivre** : entretien coûteux et infrastructures vieillissantes.
- **Amélioration des performances** : la fibre optique offre un débit plus stable et plus rapide.
- **Objectif de transition numérique** : l’ARCEP encourage le passage progressif vers des solutions plus modernes.

L’arrêt progressif du cuivre implique l’extinction des offres xDSL dans certaines zones, avec une migration vers la fibre lorsqu’elle est disponible.

## Vérifier les informations de migration dans l’espace client OVHcloud

1. Connectez-vous à votre [espace client OVHcloud](/links/manager).
2. Accédez à l’onglet **Télécom**.
3. Sélectionnez votre ligne xDSL concernée.
4. Consultez les informations de migration proposées.

Si une migration vers la fibre OVHcloud est disponible, une option de souscription vous sera proposée.

## Souscrire une offre fibre OVHcloud

### Cas 1 : Migration proposée directement dans le manager
Si votre ligne xDSL est éligible à une migration vers la fibre chez OVHcloud, vous verrez une option pour souscrire directement depuis l’espace client.

**Étapes :**
1. Suivez les instructions affichées dans l’onglet **Télécom**.
2. Validez la migration et suivez les étapes de souscription.
3. Un rendez-vous sera fixé pour l’installation de la fibre dans vos locaux.

### Cas 2 : Aucune migration proposée ou bâtiment introuvable
Si aucune migration n’est affichée ou si votre bâtiment n’est pas reconnu, procédez comme suit :

1. Rendez-vous sur [cartefibre.arcep.fr](https://cartefibre.arcep.fr/).
2. Recherchez votre adresse dans la barre de recherche.
3. Accédez à l’onglet **Déploiement fibre**.
4. Notez l’**Identifiant immeuble IPE** associé à votre adresse.
5. Contactez le support OVHcloud via un [ticket](/links/support) en précisant :
   - La référence de votre accès xDSL.
   - L’**Identifiant immeuble IPE** récupéré sur la carte de l’ARCEP.

L’équipe OVHcloud vous aidera à finaliser votre migration vers la fibre.

## Si vous ne souhaitez pas souscrire une offre fibre chez OVHcloud

Si vous ne souhaitez pas migrer vers une offre fibre OVHcloud, votre ligne xDSL sera automatiquement résiliée lors de la fermeture du cuivre.

> **Attention** : Cette résiliation est uniquement **technique**. Vous devez effectuer une **résiliation commerciale** en contactant le support OVHcloud pour finaliser la suppression de votre service et éviter toute facturation future.
