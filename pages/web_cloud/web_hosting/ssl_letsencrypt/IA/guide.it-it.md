## Procedura

**Prima di ordinare il certificato SSL Sectigo DV sul tuo hosting web**, verifica che **il nome di dominio/sottodominio** interessato dal tuo certificato SSL: 

- punti verso l'indirizzo IP del tuo hosting web. 
- sia dichiarato su uno dei siti web del tuo hosting web.
- non abbia già un certificato SSL attivo.

**Prima di completare l'installazione del certificato SSL sul tuo hosting web**, verifica che **tutti i nomi di dominio e/o sottodominio** interessati dal tuo certificato SSL: 

- puntino verso l'indirizzo IP del tuo hosting web.
- siano dichiarati su uno dei siti web del tuo hosting web.
- non abbiano già un certificato SSL attivo.

> **Prima di procedere**, verifica che **il/i nome/i di dominio e/o sottodominio/i** interessato/i dal tuo/i futuro/i certificato/i SSL:
>
> - punti/puntino verso l'indirizzo IP del tuo hosting web. 
> - sia/siano dichiarati su uno dei siti web del tuo hosting web.
> - non abbia/abbiano già un certificato SSL attivo.

> [!primary]
>
> Se desideri acquistare un certificato SSL Sectigo EV per un nome di dominio (ad esempio: `domain.tld`), verifica bene che il suo sottodominio "www" (ad esempio: `www.domain.tld`) punti anch'esso verso l'indirizzo IP del tuo hosting web e sia correttamente dichiarato su uno dei siti web del tuo hosting web.
