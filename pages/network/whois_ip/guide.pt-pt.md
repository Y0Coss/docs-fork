## Objetivo

- Deseja saber se um endereço IP é gerido ou fornecido pela OVHcloud?
- O seu domínio aponta para um endereço IP e deseja saber se este é gerido ou fornecido pela OVHcloud?
- Deseja encontrar o fornecedor que aloja o seu website através do seu endereço IP?

Para saber quem gere ou fornece um endereço IP específico, deverá efetuar um pedido WHOIS (abreviação do inglês « Who is? ») no mesmo.

**Descubra como saber se um endereço IP é gerido, fornecido e/ou associado a um serviço OVHcloud.**

## Requisitos

- Ter um endereço IPv4 (por exemplo: 203.0.113.0) ou IPv6 (por exemplo: 2001:db8:0:1b00:203:0:113:0) para o qual deseja conhecer o fornecedor ou o gestor.
- Dispor de um dispositivo ligado à Internet com um browser (por exemplo: Firefox, Chrome, Safari, Edge, etc.).

## Instruções

Aceda à ferramenta WHOIS oficial da **I**nternet **C**orporation for **A**ssigned **N**ames and **N**umbers (**ICANN**) através da seguinte ligação: [lookup.icann.org](https://lookup.icann.org/en/lookup).

Na página que aparece, insira no campo de entrada o endereço IPv4 (por exemplo: 203.0.113.0) ou IPv6 (por exemplo: 2001:db8:0:1b00:203:0:113:0) para o qual deseja conhecer o fornecedor ou o gestor.

![ICANN Lookup tool](/pages/assets/screens/other/web-tools/icann/launch-research.png){.thumbnail}

Depois de inserir o endereço IP, clique no botão azul `Lookup`{.action}.

Isto fará com que sejam apresentadas informações de endereço IP.

> [!primary]
>
> O conteúdo e a apresentação das informações no resultado de um pedido WHOIS podem diferir em função do endereço IP.

Para saber se a OVHcloud gere ou fornece o endereço IP que indicou no campo de entrada, procure, entre as informações indicadas, elementos relativos à OVHcloud (por exemplo: OVH-MNT; OVH SAS, 2 rue Kellermann, 59100 Roubaix, France; etc.)

Se for o caso, isso irá confirmar que o endereço IP para o qual efetuou um WHOIS é corretamente gerido ou fornecido pela OVHcloud.

Se necessário, procure nas informações apresentadas elementos relativos ao "registrant" (titular do endereço IP) de forma a encontrar o gestor ou o fornecedor do seu endereço IP.
