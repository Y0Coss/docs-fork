## Objectif

À la création de votre compte OVHcloud, vous avez défini une adresse e-mail de contact. Si vous êtes amené à déléguer ou partager la communication autour de votre espace client, il est possible d'ajouter des adresses e-mail de contact supplémentaires et définir des règles sur celle-ci.

**Découvrez comment gérer vos moyens de contacts et de communications avec OVHcloud.**

## Prérequis

- Disposer d'un compte OVHcloud actif.

## En pratique

Pour accéder à la gestion des communications sur votre espace client OVHcloud:

1. Connectez-vous à votre [espace client OVHcloud](/links/manager).
1. Cliquez sur votre profil dans la partie supérieure droite .
1. Cliquez sur `Mes communications`{.action}.

### Mes communications

Depuis l'onglet `Mes communications`{.action} vous retrouvez ici tous les messages qui ont été envoyés sur votre adresse e-mail de contact. Dans la partie supérieure droite du tableau, vous pouvez activer un filtre pour trier les messages par priorité.

![my messages](images/messages-01.png){.thumbnail .w-600}

### Contacts

Depuis l'onglet `Contacts`{.action} vous retrouvez l'adresse e-mail de référence du compte OVHcloud qui ne peut pas être retiré ou modifié. En plus de votre contact par défaut, vous pouvez ajouter de nouvelles adresses e-mail de contact :

- **(1)** Cliquez sur `Ajoutez un contact`{.action} à votre compte OVHcloud. Lorsque vous ajoutez un contact à votre compte OVHcloud, il est nécessaire de récupérer le code qui est envoyé sur cette adresse et revenir le saisir depuis l'espace client pour valider le contact.
- **(2)** Cliquez sur le bouton `⋮`{.action} à droite d'une adresse de contact pour afficher les options :
    - `Entrer le code de validation`{.action} : Comme évoqué précédemment, c'est ici que vous saisissez le code pour valider le contact dans votre espace client.
    - `Renvoyer le code de validation`{.action} : Vous permet de renvoyer un e-mail avec un code de validation.
    - `Supprimer`{.action} : vous permet de supprimer l'adresse de contact.

![my messages](images/contacts-01.png){.thumbnail .w-600}

### Paramètres de diffusion

Depuis l'onglet `Paramètres de diffusion`{.action}, vous allez créer des règles de triage des messages sur vos adresses e-mail de contact.

- **(1)** Cliquez sur `Créer une règle`{.action} pour appliquer une ou plusieurs règles permettant de déterminer quels contacts recevront les communications en fonction des catégories et priorités sélectionnées. 
- **(2)** Cliquez sur le bouton `⋮`{.action} à droite d'une règle pour:
    - `Activer / Désactiver`{.action} vous permet de mettre en place ou stopper une règle sans la supprimer
    - `Modifier`{.action} la règle sélectionnée
    - `Supprimer`{.action} la règle sélectionnée

![my messages](images/rules-01.png){.thumbnail .w-600}

Les règles s'appliquent selon 2 critères: 

- **La catégorie** : Compte, Facturation, incident, maintenance, produit et sécurité
- **La priorité** : Celle-ci est sur 3 niveaux, faible, moyenne et élevée.

Vous pouvez créer vos règles une par une, elles seront toutes appliquées lorsqu'un message sera transmis sur votre compte.

Vous pouvez également saisir plusieurs règles qui s'appliquent en cascade. Pour cela, lorsque vous paramétrez une règle, cliquez sur le bouton `Ajouter une condition`{.action} **(1)**. Vous pouvez en ajouter autant que nécessaire. Par défaut, si une condition s'applique, le processus s'arrête. Si vous souhaitez que le processus continu à appliquer les règles suivantes, cochez le bouton `Et continuer le traitement de règles`{.action} **(2)** sous la règle que vous avez paramétrée.

![my messages](images/rules-02.png){.thumbnail .w-600}
